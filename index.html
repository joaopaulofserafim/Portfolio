Automação de Monitoramento e NOC

Desafio: Processo manual de abertura de chamados e ausência de monitoramento centralizado, resultando em demora na resposta a quedas de link e perda de indicadores de disponibilidade.

Solução: Desenvolvimento de automação em Python que recebe eventos do PRTG. Ao detectar uma queda, o script envia alertas, abre chamados automaticamente com o provedor (via mapeamento local) e utiliza arquivos .lock para evitar duplicidade. Ao restaurar a conexão, calcula o downtime e gera relatórios quinzenais em HTML/e-mail com dados persistidos em SQLite.

Tecnologias usadas:  Python, PRTG, SQLite, SMTP, JSON, HTML

Projeto Corporativo

Provisionamento Automatizado de Identidades

Desafio: Alta demanda de criação manual de usuários no Active Directory, causando sobrecarga do time de TI, erros de digitação e demora no onboarding de novos colaboradores.

Solução: Implementação de fluxo automatizado via PowerShell que consome dados da Google Sheets API. O script valida logins únicos, cria usuários no AD, aplica políticas de segurança e associa grupos automaticamente com base no setor do colaborador, notificando as partes interessadas via e-mail.

Tecnologias usadas:  PowerShell, Active Directory, Google Sheets API v4, Service Accounts

Projeto Corporativo

Orquestrador de Liberações e Pedidos

Desafio: Sobrecarga de solicitações de liberação via chat (Financeiro), gerando falta de rastreabilidade, interrupções constantes e dificuldade na gestão de prioridades.

Solução: Desenvolvimento de ecossistema Full-stack que centraliza os pedidos. Inclui UI para gestão, API com autenticação por token para integrações externas e automação de status via triggers. O sistema notifica o solicitante em cada etapa (Recebido/Em Processo/Concluído) diretamente via Google Workspace.

Tecnologias usadas:  Google Apps Script, Google Sheets, HTML/CSS/JavaScript, REST API
